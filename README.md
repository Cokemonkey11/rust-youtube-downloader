## Rust Youtube Downloader
Youtube video downloader written in Rust

## How to use

```bash
$ git clone https://github.com/smoqadam/rust-youtube-downloader`
$ cd rust-youtube-downloader
$ cargo run -- [youtube video id]
```

for example:

`$ cargo run -- l6zpi90IT1g`
